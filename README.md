# Pure Feed for Israeli Boards (Unofficial Extension)

An ultra-lightweight, 100% client-side Chrome Extension that cleans up dynamic feed boards by filtering out commercial, agency, and promotional listings. It runs completely offline with zero data collection.

## Features
- Filters out non-private vehicle and real estate listings on dynamic feeds on the fly.
- Vaporizes embedded third-party ad containers and SafeFrames.
- Complete privacy: Makes zero network calls and stores error logs strictly inside your local browser storage.

## Manual Installation Guide (Developer Mode)
1. Download this repository as a `.zip` file and extract it.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Toggle the **Developer mode** switch in the top-right corner to **ON**.
4. Click the **Load unpacked** button in the top-left corner.
5. Select the extracted folder containing the `manifest.json` file.
